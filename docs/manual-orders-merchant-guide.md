# Guía del comercio

1. Abrir **Pedidos → Crear pedido manual**.
2. Seleccionar la sucursal propia.
3. Buscar un cliente anterior o capturar invitado.
4. Definir dirección o pickup.
5. Añadir productos, cantidades, variantes e instrucciones.
6. Guardar borrador cuando falte información.
7. Calcular y revisar precios, inventario y total.
8. Confirmar.

El comercio no puede cambiar de tenant, aplicar ajustes administrativos o asignar repartidor. Artículos personalizados y tarifa manual dependen de la configuración del negocio.
