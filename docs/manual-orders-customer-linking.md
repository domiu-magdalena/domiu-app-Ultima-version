# Vinculación de clientes

Un cliente registrado se vincula únicamente mediante su UUID después de comprobar rol, existencia y estado activo. Una dirección registrada debe pertenecer al mismo cliente.

Un invitado se guarda como snapshot sin usuario de autenticación. El teléfono sirve para operación y búsqueda humana, pero no concede lectura, propiedad ni vinculación automática. Una vinculación futura debe ser explícita y auditada.
