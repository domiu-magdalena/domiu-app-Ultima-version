# Geografía y cobertura

Cuando existen coordenadas o distancia validada, el sistema calcula tarifa y duración con la configuración activa. Sin coordenadas no bloquea por defecto: conserva dirección escrita, muestra advertencia y usa tarifa base de respaldo.

Una integración futura de Google Maps puede completar `latitude`, `longitude`, `distanceKm`, `durationMinutes` y `placeId` sin cambiar el contrato de confirmación.
