# Soporte operativo

Al escalar una incidencia incluir:

- número de pedido o borrador;
- fecha aproximada;
- panel usado;
- código de error visible;
- negocio y sucursal;
- request ID cuando esté disponible.

No solicitar contraseñas, claves privadas, datos completos de tarjeta ni copiar payloads con información personal en canales públicos.
