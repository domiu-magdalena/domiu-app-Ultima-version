# Entrega y asignación

Domicilio requiere dirección; coordenadas son opcionales. Sin distancia verificada se aplica tarifa base y se muestra advertencia. La tarifa manual exige permiso y motivo.

Pickup usa la sucursal como destino, fija domicilio en cero y no permite repartidor.

Administración puede asignar un repartidor activo, verificado, disponible y con cuenta activa. El comercio publica el pedido al flujo normal y no selecciona repartidor directamente.
