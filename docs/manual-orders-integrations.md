# Integraciones

- Supabase Auth: sesión del actor, nunca invitado.
- PostgreSQL: transacción, stock, snapshots y auditoría.
- Google Maps: coordenadas y distancia cuando están disponibles.
- Realtime: listados de pedidos existentes.
- Notificaciones: triggers actuales.
- Finanzas y liquidaciones: triggers actuales sobre `orders`.
- Vercel: APIs y UI Next.js.
