# Guía administrativa

1. Abrir **Pedidos → Crear pedido manual**.
2. Seleccionar negocio y sucursal.
3. Buscar un cliente registrado o continuar como invitado.
4. Elegir domicilio o pickup.
5. Añadir productos del negocio.
6. Configurar pago y canal.
7. Usar excepción, ajustes o asignación solo cuando exista motivo operativo.
8. Pulsar **Calcular y validar**.
9. Resolver advertencias.
10. Pulsar **Confirmar y crear pedido** una sola vez; los reintentos son idempotentes.

El administrador no puede mezclar catálogos, inventar precios o falsificar creador. El backend ignora datos no incluidos en el contrato.
