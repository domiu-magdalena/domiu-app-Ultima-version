# Matriz QA

| Área | Casos |
|---|---|
| Roles | admin, merchant, cliente/repartidor rechazados |
| Cliente | registrado, invitado, suspendido, no encontrado |
| Producto | válido, agotado, inactivo, otro tenant, variante, personalizado |
| Valores | automático, override, descuento, manipulación |
| Inventario | éxito, insuficiente, concurrencia, cancelación |
| Idempotencia | doble clic, timeout, cuerpo diferente |
| Entrega | domicilio, pickup, sin coordenadas, asignación |
| UI | borrador, resumen, errores, móvil |
