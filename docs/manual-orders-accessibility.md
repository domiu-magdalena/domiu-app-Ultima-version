# Accesibilidad

- Inputs y selects tienen etiquetas visibles.
- Botones de icono incluyen `aria-label`.
- Estados deshabilitados y carga son perceptibles.
- Contraste usa tokens de DomiU.
- El flujo funciona con teclado.
- Los errores se muestran como texto y no solo por color.
- La barra móvil mantiene acciones alcanzables.
